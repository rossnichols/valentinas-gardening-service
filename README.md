## Notes

This site is hosted with GitHub Pages as a **project site**.

Always use the trailing slash form of the URL:

- ✅ `https://rossnichols.github.io/valentinas-gardening-service/`
- ❌ `https://rossnichols.github.io/valentinas-gardening-service`

The no-slash version may be interpreted as a file path before redirecting and can break relative path resolution in some clients/caches.
